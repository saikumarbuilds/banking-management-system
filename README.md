# Employee Management System

## Overview

A RESTful web application developed using Java, Spring Boot, Spring Data JPA, and MySQL. The system provides employee management functionalities such as creating, updating, retrieving, and deleting employee records through secure and efficient REST APIs.

## Features

* Create employee records
* View employee details
* Update employee information
* Delete employee records
* Global exception handling
* Database integration with MySQL

## Technologies Used

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* Maven

## Architecture

The application follows a layered architecture consisting of Controller, Service, Repository, and Entity layers to ensure clean code organization, maintainability, and scalability.

## Learning Outcomes

* REST API development
* CRUD operations implementation
* Database integration using JPA
* Exception handling
* Maven project management
