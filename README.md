# Banking Management System

## Overview

A RESTful web application developed using Java, Spring Boot, Spring Security, Spring Data JPA, and MySQL. The system provides secure banking operations such as account creation, balance inquiry, deposits, and withdrawals.

## Features

* Create and manage bank accounts
* View account details and balances
* Deposit and withdraw funds
* Secure API access using Spring Security
* Global exception handling
* Database integration with MySQL

## Technologies Used

* Java
* Spring Boot
* Spring Security
* Spring Data JPA
* MySQL
* Maven

## Architecture

The application follows a layered architecture consisting of Controller, Service, Repository, and Entity layers to ensure clean code organization and maintainability.

## Learning Outcomes

* REST API development
* Database integration using JPA
* Authentication and authorization
* Exception handling
* Maven project management
